# 01 - Basic Data Retrieval

This folder contains exercises focusing on the fundamental building blocks of PostgreSQL.

### Concepts Covered:
* **SELECT & FROM**: Basic data extraction.
* **WHERE Clause**: Filtering records based on specific conditions.
* **ORDER BY**: Sorting data in ascending or descending order.
* **LIMIT & OFFSET**: Managing large result sets.
* **DISTINCT**: Retrieving unique values.

> *Purpose: Establishing a strong foundation for data extraction.*
