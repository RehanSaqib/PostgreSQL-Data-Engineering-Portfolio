# 02 - Relational Data Merging

This folder demonstrates how to combine data from multiple tables, a core skill for any Data Engineer.

### Concepts Covered:
* **INNER JOIN**: Retrieving matching records from both tables.
* **LEFT/RIGHT JOIN**: Handling null values and unmatched records.
* **FULL OUTER JOIN**: Comprehensive data merging.
* **SELF JOIN**: Comparing rows within the same table.
* **UNION & UNION ALL**: Combining result sets vertically.

> *Purpose: Mastering relational database relationships.*
