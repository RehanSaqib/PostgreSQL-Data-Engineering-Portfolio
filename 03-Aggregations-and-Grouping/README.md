# 03 - Data Summarization & Analytics

This folder contains queries used for generating business insights and summary reports.

### Concepts Covered:
* **Aggregate Functions**: COUNT, SUM, AVG, MIN, MAX.
* **GROUP BY**: Categorizing data for analysis.
* **HAVING**: Filtering groups based on aggregate results.
* **Math Operations**: Performing calculations within queries.

> *Purpose: Transforming raw data into meaningful statistics.*
