# 06 - Database Schema Design

This folder documents my work in database architecture, constraint management, and normalization.

### Concepts Covered:
* **DDL Commands**: CREATE, ALTER, DROP.
* **Normalization**: Practical applications of 1NF, 2NF, and 3NF.
* **Constraints**: PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL, CHECK.
* **ER Diagrams**: Mapping logical data relationships.

> *Purpose: Designing efficient and scalable database structures.*
