# 05 - Analytical Window Functions

This folder covers advanced analytical functions used for time-series and comparative data analysis.

### Concepts Covered:
* **OVER & PARTITION BY**: Defining calculation windows.
* **Ranking**: ROW_NUMBER(), RANK(), DENSE_RANK().
* **Value Functions**: LEAD(), LAG(), FIRST_VALUE(), LAST_VALUE().
* **Running Totals**: Calculating cumulative sums and moving averages.

> *Purpose: Performing high-level data analysis without collapsing rows.*
